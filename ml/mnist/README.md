# Keras
V této lekci si řekneme něco o knihovně [`keras`](https://keras.io/), která slouží jako high-level
API pro tvorbu neuronových sítí s pomocí frameworku `TensorFlow`.

## Instalace závislosti
```bash
$ pip install tensorflow
```
